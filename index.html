Youth business camp "Ocean" / Молодежный бизнес лагерь «Океан»


Mobile application Youth Business Camp "Ocean" is not commercial, but is designed to teach adolescents financial literacy. All transactions are virtual and do not carry real purchases.

Mobile application collects personal information: phone and email, specified by the user. This information is available from other authorized users.

The application uses the Internet connection to synchronize with the central database of the application. Synchronization is done as directed by the user.

The application uses access to the camera and mobile device files if the user wants to upload a photo to his profile.

The application uses local notifications to alert users of the received information when exchanging with the central database.

Мобильное приложение Молодежный бизнес лагерь «Океан» является не коммерческим, а разработанно для обучения подростков финансовой грамотности. Все совершенные сделки из виртуальных и не несут под собой реальных покупок.

Мобильное приложение собирает личную информацию: телефон и email, указанную по желанию пользователя. Данная информация доступна на других авторизованных пользователей.

Приложение использует интернет соединение для синхронизации с центральной базой приложений. Синхронизация производится по указанию пользователя.

Приложение использует доступ к камере и файлам.

Приложение использует локальные уведомления для оповещения пользователей полученной информации при обмене с центральной базой.
